Layout 1

This is the default skin of qEngine. I'm trying to create a useable skin instead of developer oriented skin as in qE 4.x.

The small boxes show modules & page menus positions:

  • SL = Site Logo.
  • TXT = Content location.
  • T1, T2, L1, L2, R1, R2, B1, B2 = Module location.
  • Usually you use T1 for banner; T2 for main menu; L1, L2, R1, R2 for more banners and/or social network feeds; B1 for bottom menu; and B2 for another banner.
  • B1 should contain only two modules. The third column is static (edit outline.tpl if needed).
